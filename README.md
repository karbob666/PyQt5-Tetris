##  💻 开发环境

- Windows 10

- Python 3.8.2

## 📦 安装依赖

```shell
pip install PyQt5
```

## 📝 功能

- 🎵 播放音乐
- ⏸︎ 暂停游戏
- 🎸 消除方块的音效
- 🖱️ 窗口自由拖拽
